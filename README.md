# ToC-Bin-Days

Town of Claremont, Western Australia Bin Days Calendar denoting whether or not its a `yellow` bin day or a `green` bin day.

Output of using CoPilot to convert the ToC PDF Calendar into json. 2025-26 Calendar published at [https://www.claremont.wa.gov.au/media/ieleld3i/25-26-waste-calendar.pdf](https://www.claremont.wa.gov.au/media/ieleld3i/25-26-waste-calendar.pdf).
